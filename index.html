關於國新

國新投資是一家專注於投資和研究、股市資訊解讀、風險控制和財務規劃。通過對各種投資機會的深入研究和分析，為客戶提供穩定的投資回報。

  • 公司簡介

    國新投資為國新集團之轉投資公司，成立於民國090年03月16日。
    我們的核心理念在於“提供增值，創造價值”，根據宏觀形勢和行業變革的趨勢，調整投資策略，追求與投資夥伴和被投資企業之間的深化合作、互相促進，助推和引領行業髮展。

    learn more

  • 沿革

    秉承價值創造和拓展，與投資夥伴之間建立互利互信的夥伴關繫，追求可持續髮展，從細節做起，儘職自律，不以高風險爲目標，髮現機會並積極創新，我們致力於為客戶實現穩定的投資回報，並與客戶建立長期的合作關係。

    learn more

  • 核心價值觀

    專業的研究和責任：我們的團隊將通過深入研究和分析市場趨勢、公司財務報表、品牌價值和行業分析來提供專業的投資建議。我們會選擇具有成長潛力的公司，並將風險控制在合理範圍內。

    learn more

  • 客戶專屬投資

    我們會根據客戶的風險偏好、財務目標和投資時間表，訂製專屬的投資方案。我們亦會與客戶保持緊密溝通，及時調整投資組合，以確保客戶的利益。

    learn more

  • 風險控制客戶至上

    我們深知投資市場存在各種風險，因此我們通過嚴格的風險管理和多元化的投資組合，來最大限度地保護客戶的資產。我們始終將客戶的需求和利益置於首位，通過貼心的服務和定期的投資建議，來幫助客戶實現其財務目標。

    learn more
